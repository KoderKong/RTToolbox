# Ricci-Notation Tensor Toolbox (RTToolbox)

Classes, unit tests, and concept demos of a tensor framework for a model-based approach to imaging and algebraic geometry with MATLAB.

Release 2026 (R2026)

## Overview

**Release 2024 (R2024):** The Ricci-notation tensor (RT) framework \[1] comprises a dual-variant index algebra, aligned with the Ricci notation of non-Euclidean geometry but designed for numeric purposes, as well as codesigned object-oriented software, called the RTToolbox for MATLAB. Compared to a numeric tensor (NT) predecessor \[2, 3, 4, 5], the RT framework enables superior ways to model imaging problems and to thereby develop solutions. Although this release of the RTToolbox did not address sparsity, unlike the prior NT framework, see below.

**Release 2026 (R2026):** This release, technically a pre-release (expect bugs, lol), adds sparse tensor functionality to the RT framework. Furthermore, it develops algebra and software for a new approach to numerical algebraic geometry, namely the any-degree unitary-triangular (Qr) factorization. The PDF slides, ReleaseNotes ([URL](https://github.com/KoderKong/RTToolbox/blob/main/ReleaseNotes.pdf)), mentions an arXiv preprint in the works. Stay tuned.
